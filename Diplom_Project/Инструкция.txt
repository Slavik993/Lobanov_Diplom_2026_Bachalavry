Как запустить приложение?

Установите зависимости:

Откройте терминал (командную строку) в папке проекта Diplom_Project и выполните команду:

bash (Команда снизу, выполняй когда находишься в папке)

pip install -r requirements.txt

Эта команда установит все необходимые библиотеки Python.
Запусти приложение
В той же папке выполни: 

bash (Команда снизу, выполняй когда находишься в папке в VSCodium)

python app.py

После запуска в терминале появится локальный адрес (обычно это что-то вроде http://127.0.0.1:7860 или http://0.0.0.0:7860).
Открой интерфейс в браузере
Перейди по адресу, который покажет терминал (чаще всего это):

http://127.0.0.1:7860

Как пользоваться интерфейсом?

В большое текстовое поле вставь историю на русском языке  
С помощью ползунка выбери количество сцен (чем больше — тем длиннее будет результат)  
(Необязательно) В поле «Style prefix» можно указать желаемый стиль, например:  Anime style  
Realistic photography  
Cyberpunk aesthetic  
Studio Ghibli style  
Dark fantasy illustration и т.д.

Нажми кнопку Generate (или аналогичную — «Сгенерировать», «Создать»)
