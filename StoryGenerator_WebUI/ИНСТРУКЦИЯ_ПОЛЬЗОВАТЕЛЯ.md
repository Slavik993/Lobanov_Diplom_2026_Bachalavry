# Руководство пользователя - StoryForge AI

## Краткая инструкция по эксплуатации

### 1. БЫСТРЫЙ СТАРТ (Windows)

**Первый запуск:**
1. Откройте папку `StoryGenerator_WebUI`
2. Двойной клик на `install.bat` - ждите завершения установки (5-10 минут)
3. Двойной клик на `start.bat` - приложение запустится
4. Откройте браузер: http://localhost:7860

**Последующие запуски:**
- Просто запускайте `start.bat`

---

### 2. ПОШАГОВАЯ ИНСТРУКЦИЯ

#### Шаг 1: Ввод сюжета
В большое текстовое поле введите вашу историю. Например:
```
Кот в костюме супергероя спасает птичку с дерева. 
Горожане аплодируют. Солнце садится за горизонт.
```

#### Шаг 2: Настройка параметров
- **Количество сцен**: от 1 до 8 (рекомендуется 3-4)
- **Стиль**: выберите из списка (Cinematic, Anime, Fantasy и др.)
- **Разрешение**: 
  - Для CPU: 256x256 (быстрее)
  - Для GPU: 384x384 или 512x512 (лучше качество)
- **Seed**: оставьте 42 или поменяйте для другого результата

#### Шаг 3: Генерация
Нажмите кнопку **"✨ Сгенерировать историю"**

#### Шаг 4: Ожидание
- **На CPU**: ~2-3 минуты на сцену
- **На GPU**: ~10-20 секунд на сцену

#### Шаг 5: Результат
После генерации вы увидите:
- Анализ сюжета
- Сгенерированные изображения
- Использованные промпты
- Кнопку для скачивания

### 3. РЕЖИМЫ РАБОТЫ

#### Для компьютеров БЕЗ видеокарты (CPU):
```
Двойной клик → start_cpu.bat
```

Или вручную:
```bash
python app.py --cpu
```

**Особенности CPU-режима:**
- Разрешение автоматически уменьшено до 256x256
- Меньше шагов генерации (15 вместо 20)
- Время генерации: 2-3 минуты на изображение
- Требования: 8GB RAM

#### Для компьютеров С видеокартой NVIDIA (GPU):
```
Двойной клик → start.bat
```

**Особенности GPU-режима:**
- Поддержка высоких разрешений (512x512)
- Быстрая генерация (10-20 секунд)
- Автоопределение GPU
- Требования: NVIDIA GTX 1060 6GB или лучше

#### Для слабых видеокарт (Low VRAM):
```bash
python app.py --low-vram
```

### 6. СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ

#### Как получить лучшие результаты:

1. **Конкретность**
   - ❌ "Кот гуляет"
   - ✅ "Рыжий пушистый кот гуляет по зеленому лугу при закате"

2. **Разделение сцен**
   - Используйте точки для разделения действий
   - Каждое предложение = потенциальная сцена

3. **Выбор стиля**
   - Sci-Fi → Cyberpunk или Cinematic
   - Сказка → Fantasy или Anime
   - Реальность → Photorealistic

4. **Количество сцен**
   - Короткий сюжет (2 предложения) → 2 сцены
   - Средний сюжет (4-5 предложений) → 3-4 сцены
   - Длинный сюжет (8+ предложений) → 5-6 сцен

5. **Seed**
   - Запоминайте понравившийся seed
   - Одинаковый seed + одинаковый промпт = одинаковый результат

### 7. РЕШЕНИЕ ПРОБЛЕМ

#### "Не запускается install.bat"
- Убедитесь что Python установлен (python.org)
- При установке Python отметьте "Add Python to PATH"

#### "Ошибка CUDA out of memory"
1. Уменьшите разрешение до 256x256
2. Закройте другие программы
3. Используйте флаг `--low-vram`
4. Переключитесь на CPU-режим: `python app.py --cpu`

#### "Генерация очень медленная"
- **Это нормально для CPU** - 2-3 минуты на изображение
- Для GPU проверьте что CUDA доступна:
  ```bash
  python -c "import torch; print(torch.cuda.is_available())"
  ```

#### "Module not found"
```bash
pip install -r requirements.txt --force-reinstall
```

#### "Модель не загружается"
- Первый запуск требует интернет (~5GB)
- Проверьте свободное место на диске (минимум 10GB)
