# Быстрый старт - StoryForge AI

## ⚡ Самый быстрый способ запустить

### Windows (рекомендуется)

1. **Установка (один раз):**
   ```
   Двойной клик → install.bat
   ```

2. **Запуск:**
   ```
   Двойной клик → start.bat
   ```

3. **Откройте браузер:** http://localhost:7860

---

## 🔧 Ручная установка (если автоматика не сработала)

### Шаг 1: Python
Убедитесь что Python 3.9+ установлен:
```bash
python --version
```

### Шаг 2: Зависимости
```bash
pip install -r requirements.txt
```

**Для CPU-режима:**
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

**Для NVIDIA GPU:**
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```

### Шаг 3: Запуск
```bash
python app.py
```

---

## 🚀 Варианты запуска

### Обычный запуск:
```bash
python app.py
```

### CPU-режим (принудительно):
```bash
python app.py --cpu
```

### Другой порт:
```bash
python app.py --port 8080
```

### Публичная ссылка:
```bash
python app.py --share
```

```bash
python app.py --low-vram
```

## ⚠️ Проблемы и решения

### "CUDA out of memory"
**Решение:** Запустите с флагом `--low-vram` или используйте CPU-режим:
```bash
python app.py --cpu
```

### "Module not found"
**Решение:** Переустановите зависимости:
```bash
pip install -r requirements.txt --force-reinstall
```

### Генерация очень медленная
**Для CPU:** Это нормально! 2-3 минуты на изображение.

**Для GPU:** Проверьте что CUDA доступна:
```python
python -c "import torch; print(torch.cuda.is_available())"
```

### Модель не загружается
**Решение:** Первый запуск требует интернет для скачивания модели (~5GB).

## 🎓 Советы для лучших результатов

1. **Конкретность:** Опишите детали сцены (цвета, освещение, объекты)
2. **Структура:** Используйте точки для разделения сцен
3. **Стили:** Экспериментируйте с разными стилями
4. **Разрешение:** На CPU используйте 256x256, на GPU 384x384 или 512x512
5. **Seed:** Запоминайте seed для воспроизведения понравившихся результатов
