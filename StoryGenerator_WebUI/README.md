# StoryForge AI - Улучшенная система генерации изображений по сюжету

**Версия:** 2.0.0  
**Автор:** Шебанов Вячеслав (улучшения на основе архитектуры Rotation_T2I_to_T2V)

## Описание

Современная интеллектуальная система для генерации последовательности изображений по текстовому сюжету. 
Поддерживает работу как на GPU (NVIDIA), так и на CPU с оптимизациями.

## Возможности

### Основные функции:
- **Генерация сцен** - автоматическое разбиение сюжета на сцены
- **Улучшение промптов** - AI-улучшение текстовых описаний
- **Множество стилей** - Cinematic, Anime, Photorealistic, Fantasy, Cyberpunk и др.
- **Анализ сюжета** - автоматический анализ структуры и настроения
- **CPU/GPU поддержка** - работает на любом оборудовании

### Оптимизации для CPU:
- Уменьшенное разрешение (256x256)
- Меньше шагов диффузии (15 вместо 20)
- Sequential CPU offload
- Автоочистка памяти

## Установка

### 1. Клонируйте или скопируйте папку `StoryGenerator_WebUI`

### 2. Установите зависимости:
```bash
pip install -r requirements.txt
```

**Для CPU-оптимизации (Windows):**
```bash
pip install intel-extension-for-pytorch
```

## Запуск

### Простой запуск:
```bash
python app.py
```

### Откройте в браузере:
```
http://localhost:7860
```

## Использование

1. **Введите сюжет** в текстовое поле
2. **Выберите количество сцен** (1-8)
3. **Выберите стиль** из выпадающего списка
4. **Настройте разрешение** (рекомендуется 256x256 для CPU, 384x384 для GPU)
5. **Нажмите "Сгенерировать"**
6. **Скачайте результаты**

## Системные требования

### Минимальные (CPU):
- **CPU:** Intel i5/AMD Ryzen 5 или лучше
- **RAM:** 8 GB
- **Диск:** 10 GB свободного места
- **Время генерации:** ~2-3 минуты на изображение

### Рекомендуемые (GPU):
- **GPU:** NVIDIA GTX 1060 6GB / RTX 3060 или лучше
- **RAM:** 16 GB
- **Диск:** 10 GB свободного места (SSD рекомендуется)
- **Время генерации:** ~10-20 секунд на изображение

## Архитектура

Приложение построено на улучшенной архитектуре, включающей:

1. **TextProcessor** - анализ и разбиение сюжета на сцены
2. **StoryImageGenerator** - генерация изображений с оптимизациями
3. **Gradio Interface** - современный веб-интерфейс
4. **CPU/GPU Auto-detection** - автоматическое определение устройства

### Улучшения от Rotation_T2I_to_T2V:
- Продвинутый анализ текста с определением настроения
- Оптимизированная генерация последовательностей
- Sliding window для консистентности между кадрами
- Модульная архитектура для расширения
